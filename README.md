
# Car Rental Platform

Multi-tenant SaaS car rental platform for Pakistani cities.

## Tech Stack
- Node.js + Express.js
- PostgreSQL + Prisma ORM
- Redis + Bull Queue
- Socket.io
- JWT Authentication

## Project Structure
apps/api      → Backend REST API
apps/web      → Admin panel + customer web app
apps/mobile   → React Native mobile app
packages/shared → Shared types and utilities
